# 사용할 라이브러리 

## JEST

* JavaScript Testing Framework.
* 통합 라이브러리.
* React-Native cli로 설치하게 되면 JEST가 자동으로 표함된다. 

## Enzyme 

* react component를 테스트할 때 사용하는 라이브러리. 
* Shallow Rendering 

## Detox

* end to end 즉, 사용자 관점에서 테스트.
* 기본적으로 모바일 앱들을 위해 개발된 라이브러리이다.
