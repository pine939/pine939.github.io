### Template Types

* **Template Type** is a special type in c++ that can take on other different types when it is initialized.
* Template Type
  * we can initialize the template using "< >"
  ```cpp
  std::vector<char> c;
  std::vector<int> i;
  std::vector<uiuc::Cube> cube;
  ```
